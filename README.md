# Personal Finance Management System (Java)

The Personal Finance Management System is a Java console application that helps users track income, expenses, and categories. This program has a balance summary that updates automatically as transactions are added. 
Users are also able to save financial data to a file of their choice and load the data back to continue adding transactions. The system has strong input validation and a menu-driven interface to make it user-friendly and error-free from data entry.

## Features

- Add Income
- Add Expenses
- View Balance Summary
- Input Validation
- Menu Interface
- Categorize transactions
- File saving/loading
